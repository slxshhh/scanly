# 🎨 Scanly Studio

**Scanly Studio** é uma plataforma profissional de design de QR Codes, construída para marcas modernas que buscam performance e identidade visual única.

**Scanly Studio** is a professional QR Code design platform, built for modern brands seeking performance and unique visual identity.

---

## ✨ Funcionalidades / Features

### 🇧🇷 Português
- **Design Studio:** Customização avançada de padrões, molduras e cores.
- **Asset Vault:** Sincronização em nuvem via Firebase para salvar e gerenciar seus designs.
- **AI Support Agent:** Assistente inteligente integrado (Gemini API) para dicas de design e suporte.
- **Exportação Profissional:** Formatos PNG, JPEG e SVG (Vetor).
- **Internacionalização:** Suporte nativo para Português (PT-BR) e Inglês (EN-US).

### 🇺🇸 English
- **Design Studio:** Advanced customization of patterns, frames, and colors.
- **Asset Vault:** Cloud synchronization via Firebase to save and manage your designs.
- **AI Support Agent:** Integrated intelligent assistant (Gemini API) for design tips and support.
- **Professional Export:** PNG, JPEG, and SVG (Vector) formats.
- **Internationalization:** Native support for Portuguese (PT-BR) and English (EN-US).

---

## 🚀 Tecnologias / Technologies
- **Frontend:** React + Vite
- **Styling:** Tailwind CSS + Motion
- **Backend:** Firebase (Auth, Firestore, Storage)
- **AI:** Google Gemini API
- **QR Engine:** QR Code Styling

---

## 🛠️ Instalação / Installation

1. Clone o repositório / Clone the repository:
   ```bash
   git clone https://github.com/slxshhh/scanly.git
   ```
2. Instale as dependências / Install dependencies:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento / Start the dev server:
   ```bash
   npm run dev
   ```
